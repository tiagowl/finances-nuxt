## Jornada do Usuário

### Jornada 1: Primeiro Acesso
1. **Descoberta** → Usuário encontra o app ao buscar "controle de gastos"
2. **Cadastro** → Cria conta com email/senha
3. **Onboarding** → Cadastra primeiras categorias de orçamento
4. **Ativação** → Cadastra primeiras despesas/receitas

### Jornada 2: Uso Diário
1. **Login** → Acessa o sistema via mobile
2. **Dashboard** → Visualiza resumo do mês atual
3. **Registrar** → Adiciona despesa ou receita do dia
4. **Categorizar** → Vincula a uma categoria
5. **Analisar** → Consulta estatísticas por categoria

### Jornada 3: Planejamento
1. **Definir metas** → Cadastra itens na lista de desejos
2. **Orçar** → Define limite máximo por categoria
3. **Acompanhar** → Monitora gastos vs orçamento
4. **Decidir** → Ajusta gastos baseado em relatórios
